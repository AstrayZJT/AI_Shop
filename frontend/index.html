分离模式启动提示

这页只关心独立前端模式，不替代 README。

1. 先启动 Spring Boot API。
2. 再在 frontend/ 目录运行 npm run dev。
3. 若后端跑在非默认端口，在 frontend/.env 中设置 VITE_BACKEND_TARGET。
4. 真模型模式下，再额外设置 SHOP_AI_ENABLED=true 和 OPENAI_API_KEY。
客户端与管理端登录态是分 scope 保存的，所以同浏览器里可以同时登录 demo 和 admin。
